# Sere Innovations - Smart Incubation Platform

A professional, high-impact landing page built for Sere Innovations, an agri-tech brand empowering small poultry farmers.

## 🚀 Key Highlights
- **Business Logic**: Includes a functional ROI Calculator and Impact Analysis.
- **Audience Focused**: Grounded messaging for rural poultry farmers.
- **Technical Polish**: Built with Next.js 16, Tailwind v4, and Framer Motion.
- **Section Compliance**: Contains all 9 required sections as per the brief.

## 🛠️ Setup
1. `npm install`
2. `npm run dev`
