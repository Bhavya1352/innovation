A high-performance, intentionally designed landing page for **Sere Innovations**—an agri-tech startup empowering small-scale poultry farmers with precision incubation technology.

## 🎯 The Mission
To translate complex agricultural technology into a usable, trustworthy, and profitable platform for rural entrepreneurs. This project avoids "vibe-coded" aesthetics in favor of a **Grounded Technical** design language.

## ✨ Core Features
- **ROI Analytics**: A dynamic profit calculator calibrated with FY25 market data.
- **Operational Impact Table**: Quantitative comparison of on-site vs. external procurement.
- **Tech Spec Grid**: Industrial-grade specification breakdown with PID control details.
- **Responsive Architecture**: Fully optimized for mobile deployments in rural environments.
- **Lead Capture**: Professional, validation-ready enquiry system.

## 🛠️ Tech Stack
- **Framework**: Next.js 15 (Turbopack)
- **Styling**: Tailwind CSS v4 (Alpha)
- **Icons**: Lucide React
- **Motion**: Framer Motion (Subtle transitions)

## 📖 Project Structure
- `src/components/`: Modular, "de-vibed" UI components.
- `src/app/globals.css`: Custom design tokens for the Sere design system.
- `implementation_note.md`: Detailed documentation on design rationale and assumptions.

## 🚀 Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Build for production: `npm run build`
